収集する情報

位置情報

本アプリは、登録したバス停への接近を検知するために位置情報（GPS）を使用します。位置情報はお使いのデバイス上でのみ処理され、外部サーバーに送信・保存されることはありません。

個人情報

本アプリは、氏名・メールアドレス・連絡先などの個人情報を一切収集しません。

第三者サービス

本アプリは発車時刻データの取得に Transitland（transitland.org）のAPIを使用しています。APIリクエスト時には停留所IDおよび日時情報が送信されます。

データの保存

アプリの設定・登録停留所・発車時刻キャッシュはすべてお使いのデバイス内にのみ保存されます。クラウドへの同期や外部への送信は行いません。

お問い合わせ

プライバシーに関するご質問は GitHub Issues までお問い合わせください。

English

Last updated: March 28, 2026

Information We Collect

Location Data

This app uses GPS location to detect when you approach registered bus stops. Location data is processed only on your device and is never transmitted to or stored on external servers.

Personal Information

This app does not collect any personal information such as your name, email address, or contacts.

Third-Party Services

This app uses the Transitland (transitland.org) API to retrieve transit departure data. Stop IDs and date/time information are sent with API requests.

Data Storage

All app settings, registered stops, and departure time cache are stored locally on your device only. No data is synced to the cloud or transmitted externally.

Contact

For privacy-related inquiries, please contact us via GitHub Issues.
